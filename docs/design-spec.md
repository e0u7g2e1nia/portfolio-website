# 作品集网站设计需求文档

## 一、整体定调

**气质：** 淡、简、梦幻、有质感。克制但不冷漠，有细节但不堆砌。

**色调：** 低饱和柔和渐变——冷蓝灰、暖粉灰、薄荷绿、米白。参考色 #e1e6ef 和 #f1e5e7。
背景随滚动位置渐变切换，渐变方向从左上到右下。
各内容区块对应不同色相，但整体保持相近的低饱和度和明度，仅色相作轻微偏移（如蓝灰→粉灰→绿灰→米白），不要出现饱和度或明度突变，过渡平滑自然。具体实现方式：以 #e1e6ef 的 HSL 参数为基准，保持 S（饱和度）和 L（亮度）数值不变，仅在各区块之间调整 H（色相），产生柔和的色相漂移效果。

**字体方向：** 衬线大标题 + 小号无衬线正文，主要为中文，中英混排。

## 二、交互系统

### 光标

参考图1中的光标小柔和圆形效果。替换默认鼠标为自定义光标：一个小圆形，模糊边缘、有颗粒感，从中心 #ffca79（100% 不透明）向外渐变为 #ffd058（0% 不透明），呈现柔和发光的浮动圆点效果。悬停在可点击元素上时，光标轻微放大并有颜色变化的状态反馈。

### 导航

固定顶端右侧导航栏，导航项依次为：**关于我 / AI共创 / 产品 / 设计 / 自媒体**。

点击任意导航项，展开导航面板，面板内以缩略图网格展示该分类下的所有项目。点击任意缩略图进入对应项目的详情页。

## 三、页面结构

### 第一屏 — Hero

页面中央有一条从左到右的弯曲轨道线，线条风格参考提供的参考图2中的曲线（柔和曲线，有质感但不粗重）。

**轨道线上方（不遮挡线）：**

- 大标题："吴沂铮 Eugenia Wu"
- 大标题同高度右侧：个人照片，矩形或自然边缘裁剪，风格与整体气质匹配

**轨道线下方（不遮挡线）：**

- 小字两行正文：
  - "复旦大学 · 社会学 · 具备创意的产品思维和审美方式"
  - 社交媒体账号图标附加链接、GitHub 图标附链接。排版简洁，与整体气质统一。

---

### 项目详情页

每个项目有独立完整页面，结构统一如下：

- **最顶部：** 返回按钮（返回主页或上一页）
- **上部：** 项目名 + 背景说明文字
- **主体：** 与该项目在内容区块中的展示形式保持一致——即在主页怎么展示，详情页就以相似的方式呈现更完整的内容（更多截图、更完整视频、更多说明文字）
- **底部：** 可选的访问链接或下载按钮

---

## 四、移动端适配

移动端尽量保持与桌面端一致的视觉效果和气质。

轨道线交互降级方案：悬停改为点击触发 tooltip，点击第二次进入详情页。其余交互（3D翻转、横向画廊滑动）在移动端保持，手势操作替代鼠标操作。
